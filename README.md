# js-unconf-logo-polymer

## Usage

```html
<js-unconf-logo animation="show-back-x"></js-unconf-logo>
```

## Attributes

```
animation             string

  The element can have the values:

  show-front
  show-back-y
  show-top
  show-bottom
  show-back-x
  show-left
  show-right
```
